# 스냅노트 (SnapNote)

> 사용자의 에너지를 아껴주는 메모 기반 데이터 수집/정리 도구

## 소개

스냅노트는 최소한의 노력으로 정보를 수집하고 자동으로 정리해주는 개인 지식 관리(PKM) 도구입니다. 복잡한 설정이나 학습 없이도 직관적으로 사용할 수 있으며, 로컬 우선 저장 방식으로 개인정보를 안전하게 보호합니다.

### 주요 기능

- **원클릭 스크래핑**: Alt + 클릭으로 텍스트와 링크를 즉시 저장
- **직관적인 북마크 시스템**: 7가지 상태 아이콘으로 정보 관리
  - 미확인 (빈 체크박스)
  - 확인 완료 (체크박스)
  - 제외/거절 (X 체크박스)
  - 진행중 (동그라미)
  - 주의/중요 (세모)
  - 취소/폐기 (X)
  - 즐겨찾기 (별)
- **자동 정리**: 시간순 정렬과 북마크별 필터링
- **빠른 검색**: 수집한 모든 정보를 즉시 검색

### 차별점

1. **최소한의 노력**
   - 클릭 한 번으로 정보 저장
   - 직관적인 상태 관리
   - 자동화된 정리 시스템

2. **로컬 우선**
   - 개인정보 보호 중시
   - 오프라인 작업 가능
   - 빠른 응답 속도

3. **심플한 디자인**
   - 깔끔한 사용자 인터페이스
   - 눈의 피로도를 줄이는 컬러 시스템
   - 라이트/다크 테마 지원

## 대상 사용자

- **디지털 노마드**: 다양한 온라인 정보를 효율적으로 수집하고 관리해야 하는 사용자
- **지식 노동자**: 연구, 기획, 분석 등을 위해 많은 자료를 수집하고 정리하는 전문가
- **학생/연구자**: 학습과 연구를 위한 자료를 체계적으로 관리해야 하는 사용자
- **콘텐츠 크리에이터**: 다양한 레퍼런스와 아이디어를 수집하고 활용하는 창작자

## 기술 스택

- Frontend: Vite + React + TypeScript
- Styling: UnoCSS
- State Management: Zustand
- Desktop: Tauri + Rust

## 개발 환경 설정

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build
```
